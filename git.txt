Шпаргалка для работы с командной стракой:
0) "~" - Корневая папка пользователя
1) "cd" - переход в папку
2) "mkdir имя папки" - Создание новой папки
3) "cd имя папки" -Переход в папку
4) "echo Some text > file.txt" - Создание файла с тескстом
5) "ls" - Список файлов в папке
6) "cat file.txt" - Чтение файла
7) "rm file.txt" - Удаление файла
8) "pwd" - Полный путь к папке
9) "ls" - Отображение содержимого в папке
10) "ls -la" - (если git bash) - Отображение скрытых файлов
    "ls -Force" - (если PowerShell) - Отображение скрытых файлов
11) "pwd" - путь к текущей папке

Шпаргалка команд git:
1) "git add имя файла" - добавление изменений файла в git index(сохранение файла)
   "git add ." - добавление всех изменений файлов в git index(сохранение файла)
2) "git commit" - сохранить изменения в репозитроий
3) "git checkout" - перейти к определённой версии
4) "git status" - Отображает текущее состояние GIT репозитроия
5) git commit -m "message" - Создание комита с записью изменений в репозитроий
6) git log - Просмотр истории изменений (коммитов)
7) git checkout <comit hash> - перейти к определённой версии проекта по SHA1 хэшу коммиту
8) git checkout <branch name> - Переход в определённую версию проекта по названию ветки
9) git cat-file -t name hash - Прочитать содержимое обьектов git по определённому хзшу
10) git cat-file -p name hash - Прочитать содержимое коммита по определённому хзшу
